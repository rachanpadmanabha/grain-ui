Dark mode

Grain ships dark tokens out of the box and respects prefers-color-scheme. If you want an explicit app-level toggle, set data-theme yourself.

Automatic

@media (prefers-color-scheme: dark) {
  :root {
    --gr-bg: hsl(0 0% 9%);
    --gr-surface: hsl(0 0% 11%);
    --gr-text: hsl(0 0% 94%);
  }
}

Explicit override

html[data-theme="dark"] {
  --gr-bg: hsl(0 0% 9%);
  --gr-surface: hsl(0 0% 11%);
  --gr-text: hsl(0 0% 94%);
}

Example theme snippets

Here are four starting points you can drop into your own app and tune from there.

Warm / earthy

:root {
  --gr-hue: 28;
  --gr-accent: hsl(28 75% 44%);
  --gr-accent-hover: hsl(28 75% 38%);
  --gr-accent-light: hsl(28 90% 93%);
}

Cool / corporate

:root {
  --gr-hue: 214;
  --gr-accent: hsl(214 84% 46%);
  --gr-accent-hover: hsl(214 84% 40%);
}

High contrast

:root {
  --gr-bg: #fff;
  --gr-text: #000;
  --gr-border: #000;
  --gr-accent: #000;
  --gr-text-inverse: #fff;
}

Pastel

:root {
  --gr-hue: 318;
  --gr-accent: hsl(318 68% 58%);
  --gr-accent-light: hsl(318 100% 96%);
  --gr-radius: 16px;
}
